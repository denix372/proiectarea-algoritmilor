# 📌 Divide & Conquer & Binary Search — Complete Roadmap

While Binary Search is technically a specific application of D&C (where you "divide" the problem into two halves but only "conquer" one), merging them into a single study path is a genius move. This allows you to master $O(\log N)$ and $O(N \log N)$ logic simultaneously.
